Wir nutzen **Obsidian** als unser "externes Gehirn". Es ist ein mächtiges Wissensmanagement-Tool, das auf lokalen **Markdown-Dateien** basiert. Das bedeutet: Alles, was wir schreiben, gehört uns, ist offline verfügbar und wird über Git versioniert.

## Erste Schritte

1. **Installation:** Lade Obsidian von [obsidian.md](https://obsidian.md/) herunter.
    
2. **Vault öffnen:** Wähle beim Start "Open folder as vault" und wähle diesen Ordner (`01_Obsidian`) aus.
    
3. **Verlinken:** Das Herzstück von Obsidian sind Verknüpfungen. Tippe `[[Dateiname]]`, um eine Notiz mit einer anderen zu verbinden.

4. Lade das "Obsidian Git" plugin herunter!

## Markdown Crashkurs

Markdown ist eine einfache Art, Text zu formatieren, ohne die Hände von der Tastatur zu nehmen:

| Ziel            | Tippweise                              | Ergebnis                |
| --------------- | -------------------------------------- | ----------------------- |
| **Überschrift** | `# Titel` oder `## Untertitel`         | Strukturierte Header    |
| **Fett**        | `**Text**`                             | **Text**                |
| **Kursiv**      | `*Text*`                               | _Text_                  |
| **Liste**       | `* Punkt 1`                            | • Punkt 1               |
| **Checkliste**  | `- [ ] Aufgabe`                        | [ ] Aufgabe             |
| **Code**        | `` `inline` `` oder ` ``` ` für Blöcke | `code`                  |
| **Verlinkung**  | `[[Andere Notiz]]`                     | Link zur internen Seite |
## Empfohlene Arbeitsweise für unser Team

- **Kein Chaos:** Speichere Bilder/Anhänge immer im dafür vorgesehenen Unterordner (Attachments).
    
- **Atomare Notizen:** Schreibe lieber viele kleine, spezifische Notizen (z.B. "Motor-Kalibrierung") als eine riesige, unübersichtliche Datei.
    
- **Tags:** Nutze `#Hardware`, `#Software` oder `#Meeting`, um Themen schnell über die Suche zu finden.
    
- **Git-Integration:** Bevor du deine Arbeit beendest, mache einen `git push`, damit deine Notizen für alle verfügbar sind.
